## Asset Selection Guide by Post Type

### Welcome / Brand
Core: `Green BG@4x.png`, `text-lets-fuel-learning.png`
Decor: `sparkle-stars-3d-yellow.png`, `blob-3d-light-blue.png`, `plus-3d-blue.png`, `triangle-3d-blue-left.png`

### Educational / Tip
Core: Logo (small), large headline text
Decor: Letter assets (a/b/c/e), `star-yellow-3d.png`, `plus-3d-yellow.png`, `rounded-square-3d-green.png`
Extra: `taleemabad-student-icon.png`

### Announcement / Launch
Core: `taleemabad-logo-green-shadow.png`, bold headline
Decor: `triangle-outline-3d-orange.png`, `oval-yellow-3d.png`, `diamond-3d-purple.png`, `star-corner-3d-yellow.png`

### App Promotion / Download
Core: `taleemabad-app-icon.png`, `taleemabad-tablet-mockup.png`, `btn-download-now.png`
Decor: `plus-3d-blue.png`, `sparkle-stars-3d-yellow.png`, `oval-green-3d.png`

### Multilingual / Language
Core: Logo + language text assets (`text-english-white.png` etc.)
Decor: Urdu letter assets, `letter-a-3d-orange.png`, `urdu-question-mark-3d-blue.png`

### Team / Hiring
Core: Logo + `text-product-experts.png` or `text-relationship-experts.png`
Decor: `location-pin-blue.png`, `pill-3d-purple.png`, `diamond-shape-white.png`
